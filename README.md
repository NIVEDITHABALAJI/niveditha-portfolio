# 🌸 Niveditha B — Developer Portfolio

A creative, pastel-aesthetic personal portfolio built from scratch with vanilla HTML, CSS & JavaScript — showcasing my work as a Full Stack Developer.

### 🔗 Live Demo
[View Live Portfolio](https://nivedithabalaji-portfolio.vercel.app/) 

---

## 📖 About

This portfolio highlights my journey as a Full Stack Developer skilled in the MERN stack, AI integration, and UI/UX design. It includes my education, technical skills, projects, and certifications — all presented in a soft pastel, animated, multi-section design.

## ✨ Features

- 🎨 Soft pastel aesthetic with custom color palette
- 🖱️ Custom animated cursor (desktop)
- ⌨️ Typewriter effect for role titles
- 🌊 Smooth scroll-reveal animations on every section
- 📱 Fully responsive — mobile, tablet & desktop
- 🗂️ Filterable projects section (All / Featured / Other)
- 📊 Animated CGPA progress bars in the Education timeline
- 📬 Interactive contact form with success state
- 🍔 Mobile hamburger navigation menu

## 🛠️ Built With

- **HTML5** — semantic structure
- **CSS3** — custom properties, Flexbox, Grid, animations
- **JavaScript (Vanilla)** — DOM interactivity, IntersectionObserver, typewriter effect
- **Google Fonts** — Playfair Display & DM Sans

## 📂 Sections

1. **Home** — Intro, typewriter role animation, social links
2. **About** — Bio, highlights, quick stats
3. **Education** — Animated timeline (MCA & B.Sc.)
4. **Skills** — Categorized skill chips (Frontend, Backend, AI/APIs, Tools, etc.)
5. **Projects** — AethelForge, Nudge, LuminaNest, CircleUp, Cream & Co.
6. **Certifications** — 6 certifications with issuing organizations
7. **Contact** — Contact form + direct contact details

## 🚀 Getting Started

Clone the repo and open `index.html` in your browser — no build steps required:

```bash
git clone https://github.com/NIVEDITHABALAJI/niveditha-portfolio.git
cd niveditha-portfolio
```

Then open `index.html` with **Live Server** in VS Code, or simply double-click the file.
